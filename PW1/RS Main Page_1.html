Conceito

Este espaço web foi desenvolvido por três alunos do 2º ano do Curso de Tecnologias e Sistemas de Informação para a Web no âmbito do projeto da cadeira de Programação Web I.

Foi explorada uma vertente em que os alunos sentiram que necessitava desenvolvimento, a reserva de salas apresentava um sistema pouco prático, daí tentarem criar um espaço onde era apresentada de forma organizada as informações de cada sala de forma a facilitar a escolha do utilizador.
